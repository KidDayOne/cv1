Relative Size

I believe using relative styling is preferable to other styling methods. The "%","em" and "rem" units are useful in making a website more responsive because making the size relative will allow any type of browser to render the elements in the same way.
For example let a given elements' size be set to "absolute", then when the page is zoomed in/out or minimized/maximized, the element will either be distorted or bent out of shape. However, by using the relative units, the elements will grow and shrink according to their parent containers.

CSS 3

CSS3 is the latest version of the Cascading Style Sheet language. It brings a lot of new features and additions to help developers to create great styling. Some major enhancements that have been included are:

    Media queries are perhaps one of the most revolutionary modules, responsible for the responsive design that has become standard in todayâ€™s web world. They can be used to target specific media for the "style", "link", "source", and other HTML elements with the media= attribute. They also apply styles with the "CSS @media" and "@import at-rules"

    CSS3 brings with it new ways of describing color. The color keyword list has been extended to include 147 colors and other options:
  • RGBA : the same as RGB only with a 4th value; alpha for the transparency level.
  • HSL and HSLA : HSL stands for hue, saturation and lightness. Unlike RGB, this can tweak either just the saturation or the light while keeping the same base hue...the "a" is used the same way as it was used in rgba.

    In CSS3, there is no need to include an element for every background image. It allows us to add more than one image to any image. The structure is as follows:
  • "background-image: url(firstImage.jpg), url(secondImage.jpg)...;"

    W3C added the syntax for generating gradients too. The Gradient datatype is a special type of "image" that consists of a progressive transition between two or more colors. There are Linear Gradients and Radial Gradients.
  • Linear Gradients : in this type, colors proceed along a straight axis.
  • Radial Gradients : are circular or elliptical gradients, colors blend out from a starting point in all directions.
