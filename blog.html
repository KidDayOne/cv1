Relative Size

I believe using relative styling is preferable to other styling methods. The "%","em" and "rem" units are useful in making a website more responsive because making the size relative will allow any type of browser to render the elements in the same way. For example let a given elements' size be set to "absolute", then when the page is zoomed in/out or minimized/maximized, the element will be either be distorted or bent out of shape. However, by using the relative units, the elements will grow and shrink according to their parent containers.
